Memory Box Bébé

La Memory Box Bébé, c'est le cadeau de naissance dont rêvent les jeunes parents : une jolie boite à souvenirs et des crédits photos (100 ou 300 tirages photos) valables 1 an pour la remplir. Le premier biberon avec Papa, le premier bain, la première nuit dans sa chambre... la Memory Box Bébé est un coffret parfait où ranger tous ces bons souvenirs !
